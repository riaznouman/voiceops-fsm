# VoiceOps

AI-Powered Voice Receptionist and Integrated Field Service Management System

## Project Overview

VoiceOps is a capstone project for COIT20273 (Software Design and Development Project) at CQUniversity Australia. The system combines an AI-powered voice receptionist with a field service management platform to automate inbound call handling, job scheduling, technician dispatching, and work order management.

## Tech Stack

- **Backend & Admin Panel:** Next.js 15 (App Router, Server Components, API Routes)
- **Mobile App:** Expo SDK 55 with React Native
- **Database:** PostgreSQL with Prisma ORM (PostGIS for location queries)
- **AI Voice Integration:** Vapi.ai
- **Authentication:** NextAuth.js (JWT)
- **State Management (Mobile):** Redux Toolkit / RTK Query
